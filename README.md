# CSS MIN HEIGHT PERCENTAGES

  Mobile-first classes for css-min-height-percentages.
  Set the desired css-min-height-percentages on any element for any breakpoint.
  Base class names are namespaced across three breakpoints:

*  -ns = not-small (covers everything larger than mobile)
*  -m  = medium
*  -l  = large

## Install
```
npm install --save-dev css-min-height-percentages
```
or download the css on github and include in your project.
